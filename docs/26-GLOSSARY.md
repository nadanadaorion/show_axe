# Glossary

**Assignment** — One physical unit inside a grouped Equipment line, with a free-form use.

**CH** — Input channel label/number stored on an Input List row.

**Equipment line** — Grouped Show entry such as `2 × Shure SM58`.

**Generated row** — Input List row linked to an Equipment assignment through provenance IDs.

**Input List** — Ordered technical list of sources, equipment, channels, phantom, patch, and notes.

**Library** — Reusable catalogs for Equipment, People, Categories, Roles, Types, and Origins.

**Lock** — Temporary remote claim that prevents a second online device from editing a Show.

**Manual row** — Input List row created directly without Equipment provenance.

**Monitor return** — Wedge/IEM/sidefill/drum-fill output assigned to a destination.

**Origin** — Provider or source of Equipment, such as own, venue, rental, or artist.

**Preset** — Reusable snapshot used to merge into or replace operational Show sections.

**Public slug** — Stable random value used in a Show's public route.

**Revision** — Server-side optimistic concurrency number.

**Show** — Main operational snapshot containing all preparation information.

**Snapshot** — Copied independent data that no longer changes when its source Library/Preset changes.

**Workspace** — Shared Library, Presets, and Preferences document in V2.
